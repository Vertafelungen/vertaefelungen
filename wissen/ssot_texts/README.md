# SSOT Include-Texte

Dieser Ordner enthält Langtexte für SSOT-CSV-Felder als Markdown-Dateien.

## Regel
- In CSV-Zellen steht statt Langtext eine Referenz im Format `@include:<relpath>`.
- `<relpath>` ist **relativ zu `wissen/ssot_texts/`**.
- Include-Root ist ausschließlich `wissen/ssot_texts/`.

## Konvention
- Langtexte gehören in Dateien unter `wissen/ssot_texts/`.
- In den CSV-Dateien sollen keine langen Fließtexte mehr direkt gepflegt werden.
