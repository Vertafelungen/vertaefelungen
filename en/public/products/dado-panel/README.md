# Half-Height Paneling (Wainscoting)

This folder contains all documented models of our half-height paneling systems – also referred to as “wainscoting”. These panels typically reach up to window sill height or mid-wall level and are ideal for hallways, libraries, or representative wall areas.

Each Markdown file includes details such as:

- Historical inspiration or origin
- Available wood types (e.g., pine or oak)
- Versions (kit / fully assembled)
- Dimensions and typical construction
- Example images and applications
