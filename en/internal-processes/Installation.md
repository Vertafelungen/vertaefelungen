# Installation Guide for Wood Paneling

This guide provides an overview of the typical installation process for traditional or historic-style wall paneling.

## 1. Preparation
- Check wall: stable, dry, and level
- Compare site measurements to layout plan
- Consider positions of outlets, radiators, etc.

## 2. Tools & Materials
- Level, tape measure, miter saw, drill
- Mounting adhesive, screws, hidden fasteners if used
- Installation plans, panels, moldings, and profiles

## 3. Installation Process
1. **Fix horizontal rail:** Attach at intended base height  
2. **Install panels:** Work symmetrically from the center outward  
3. **Apply moldings:** Use miter cuts and adhesive  
4. **Create cutouts:** For sockets or structural recesses  
5. **Final check:** Clean glue residues, check fit and finish

## 4. Surface Finish
Panels may be pre-treated (oiled, lacquered, patinated) or finished after installation.

## 5. Heritage Project Notes
- Ensure reversibility (do not glue to historic plaster)
- Coordinate with preservation authorities

## 6. Safety
- Electrical work must be done by professionals
- DIY installation is done at customer's own risk
